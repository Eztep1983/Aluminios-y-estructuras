# **App Name**: Aluminios y Estructuras

## Core Features:

- Project Carousel: Display a carousel of the latest 7 projects fetched in real-time from Firestore. Each project card includes an image and description. Users can swipe left/right to navigate and view all projects on the 'Nuestro Trabajo' page.
- Responsive Project Grid: Display all projects on the 'Nuestro Trabajo' page in a responsive grid layout (3 columns on desktop, 2 on tablet, 1 on mobile).
- Project Filtering: Implement category filters on the 'Nuestro Trabajo' page (Puertas, Portones El√©ctricos, etc.) to filter the projects. Categories are optional field in Firestore.
- WhatsApp Contact: A prominent WhatsApp button to initiate a chat with a pre-defined message. Includes direct call option. The tool should attempt to place the URL from the original request if found.
- Testimonials Section: Display user testimonials with an option for users to submit their own testimonials anonymously or with their name. User generated content is to be stored to Firestore.
- Google Maps Integration: Embed a Google Maps component showing the establishment's location.
- AI Powered Image Optimization: Leverage AI powered tool to optimize images hosted in Cloudinary for faster loading times and better SEO.

## Style Guidelines:

- Primary color: White (#FFFFFF) to give a clean and modern look.
- Secondary color: Shades of Blue (#2980b9, #2471A3) to represent trust and stability.
- Accent color: A lighter shade of blue (#85C1E9) for interactive elements like buttons and highlights.
- Headline font: 'Roboto' sans-serif for titles, providing a modern and readable appearance.
- Body font: 'Open Sans' sans-serif for main text, ensuring clarity and readability.
- Use clear, modern, and minimalist icons in shades of blue to represent project categories and contact options.
- A clean, grid-based layout with a focus on visual presentation of projects, ensuring a professional and organized look.
- Subtle transitions and animations using blue tones to improve user experience, especially in the project carousel.
